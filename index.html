There are different programming languages that involve web development

HTML

HTML is a markup language. It is used to define the layout or the structure of a webpage.

CSS

CSS is a styling language. It is used to style the components of a HTML file.

Javascript

Javascript is a programming language used to add interactivity in the website. It makes the website dynamic.
